## ⚠️ IMPORTANT

### **Before Deploying:**

1. **Update Formspree Form ID**
   - Get ID from formspree.io after signup
   - Replace `xyzqwert` in index.html line ~525
   - WITHOUT this, contact form won't work

2. **Update Domain References** (if using custom domain)
   - Update canonical URL in head
   - Update og:url and twitter:url
   - Update sitemap location if needed

3. **Test Locally**
   - Open index.html in browser
   - Test all links
   - Test form submission (if Formspree ID is set)
